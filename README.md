# MatchPulse Bot

AI-powered sports analytics and match insights bot for Telegram.

## Commands

- `/start` - Welcome and menu
- `/matches` - Upcoming matches
- `/live` - Live scores
- `/match <id>` - Match details
- `/team <name>` - Team analytics
- `/predict <match>` - AI prediction
- `/subscribe` - Get daily insights

## Installation

```bash
pip install -r requirements.txt
python -m bot.main
